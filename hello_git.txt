How to add an existing VS Code project to git and github
& How to do commit & push whenever changes happen

1. create a repository on github
2. copy its url - we'll need it on vs code
3. vsc (vscode): open project folder
4. vsc: click on source control icon on left bar
5. vsc: click on "+" icon on top & select project folder
6. source control will show the number of files to commit
7. enter message/description of what the commit is about
8. click the check mark on top to commit the changes
9. command palette: git add remote (if github repository hasn't been added)
10. push the committed changes to github: source control "..." -> push or icon at bottom bar
11. check changes on github repository


How to clone from github to another computer

12. copy project url from github
13. vsc view/command palette... git clone
14. enter project url
15. select floder where to store the local repository


How to remove local git repository

16. vsc view/command palette... git close repository
17. delete hidden folder .git from project local folder
